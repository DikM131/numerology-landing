# 00 READ FIRST

Вводный документ по проекту Numerolog. Эта документация описывает текущую архитектуру, принятые решения, правила разработки и сопровождения проекта. Все разделы взаимосвязаны и должны читаться по мере необходимости. Проект представляет собой премиальный лендинг на React + Vite + Tailwind CSS с акцентом на производительность, анимации и удобство сопровождения.
